# Kabinet Dharma Sangha - Forum Beasiswa

Folder ini berisi versi HTML + CSS + JavaScript statis dari desain website Forum Beasiswa dengan nuansa visual yang terinspirasi dari referensi Y2K editorial.

## Isi file
- `index.html`
- `styles.css`
- `script.js`

## Jalankan lokal
Cukup buka `index.html` di browser.

## Deploy ke GitHub Pages
1. Buat repository baru di GitHub.
2. Upload semua file dari folder ini ke branch `main`.
3. Buka `Settings` > `Pages`.
4. Pada **Build and deployment**, pilih:
   - **Source**: Deploy from a branch
   - **Branch**: `main` dan folder `/root`
5. Simpan, lalu tunggu link GitHub Pages muncul.

Contoh URL hasil publish:
`https://usernamekamu.github.io/nama-repo/`
